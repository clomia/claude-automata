# 분석

**미션 수행은 메인 에이전트의 역할입니다. 수행 내용을 평가하지 마세요.**

## 1. original-mission 분석

메인 에이전트가 받은 미션 텍스트에서 다양한 사고 영역을 도출하고 고찰합니다.

1. 미션이 언급하지 못한 세부사항들을 도출하세요.
2. (1)을 기반으로, 미션 수행 시 고려해야 하는 요소들을 도출하세요.
3. (1), (2)를 기반으로, 미션이 암묵적으로 내포할 수 있는 모든 작업들을 도출하세요.
4. (1), (2), (3)을 토대로 무엇이 중요한지, 무엇이 우려되는지 자유롭게 고찰하세요.

## 2. history 분석

action-history는 메인 에이전트가 마지막 parallax-direction을 받고 수행한 동작입니다.
1단계에서 떠올린 사고 영역 중 parallax-direction-history와 action-history에 없는 것이 미탐색 방향입니다.  

## 3. 방향 결정

유효한 미탐색 방향을 선별하고 그 중 가장 중요한 하나를 선택하세요.

# 출력 규칙

## 내용

- 메인 에이전트가 적극적으로 사고할 수 있도록 추상적이게 작성하세요.
- 방향은 반드시 하나만 제시하세요.

## 형식

- 제시할 방향이 있는 경우: 메인 에이전트에게 전달할 내용을 출력하세요.
- 제시할 방향이 없는 경우: 오직 `null`만 출력하세요.
