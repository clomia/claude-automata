---
name: translate-prompt
description: 한국어 프롬프트(.ko.md)를 영어 버전(.md)으로 번역합니다.
argument-hint: "[filepath.ko.md]"
disable-model-invocation: true
---

1. `$ARGUMENTS` 파일을 읽고 내용과 연관된 파일들을 탐색하여 프롬프트의 의도를 이해합니다.
2. 같은 디렉토리에 `.ko.md`를 `.md`로 바꾼 경로의 영어 버전 파일이 이미 존재하면 삭제합니다.
3. 이해한 내용을 기반으로 영어 버전을 작성하여 해당 경로에 저장합니다.
   - 한국어 버전에 담긴 의미와 미묘한 정보들이 변하지 않도록 충분히 고민하면서 번역해야 합니다.
4. 번역 과정에서 한국어 버전에 담긴 의미가 손실되진 않았는지 검토합니다.
   - 이슈가 있으면 개선 후 다시 검토합니다. 이슈가 발견되지 않을 때까지 반복합니다.
